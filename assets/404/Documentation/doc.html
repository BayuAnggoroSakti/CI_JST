Installation on your server via FTP

How To Install RPZ Via FTP :
Step 1 - Log into your hosting space via an FTP software (Filezilla for example).
Step 2 - Unzip your Download Pack file and use the extracted RPZ theme folder.
Step 3 - Upload the extracted RPZ theme folder into one folder or at the root of your server.
